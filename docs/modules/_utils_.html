range

  • range(ranges: string): string
  • range(start: string, end: string): string
  • range(start: string, end: undefined, processedEscapes: true): string
    • Defined in utils.ts:22

    Generates a string containing all the unicode characters within the provided regex-style ranges, inclusive. Supports special escape characters "\-" as literal dash (-), and "\\" as literal backslash ("\")

    example
    range("a-c"); // "abc"
range("d-fkw-z"); // "defkwxyz"
range("0-3a-fQZA-C"); // "0123abcdefQZABC"
range("Hello, world \\- Obama \\\\ \\\\-^ a-c"); // "Hello, world - Obama \\ \\]^ abc"

    Parameters

    • ranges: string

      A string containing one or more regex-style ranges, specifying the characters that should be in the return string.

    Returns string
